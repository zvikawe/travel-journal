# Journeys — Zvika & Amit

A photography-led travel website inspired by modern editorial destination guides.

## Publish

1. Create a public GitHub repository, such as `travel-journal`.
2. Upload every file and the `data` folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/(root)`.
6. Save.

## Update an existing repository

Delete or replace the old files with the files from this package. Keep the folder structure:

- index.html
- trip.html
- styles.css
- script.js
- trip.js
- data/trips.js

The photographs are loaded from Unsplash, so the website needs internet access when viewed.
